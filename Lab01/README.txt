3. HƯỚNG DẪN BIÊN DỊCH VÀ CHẠY CHƯƠNG TRÌNH

Cách 1: Chạy trên phần mềm NetBeans (Khuyên dùng)

Bước 1: Mở NetBeans, chọn File -> Open Project và chọn thư mục dự án QuanLySinhVien.

Bước 2: Tìm file MainApp.java trong thư mục Source Packages -> quanlysinhvien.

Bước 3: Nhấn chuột phải vào file MainApp.java và chọn "Run File" (hoặc nhấn phím tắt Shift + F6).

Cách 2: Chạy bằng dòng lệnh (CMD) Nếu không dùng NetBeans, bạn có thể chạy bằng cửa sổ dòng lệnh (Command Prompt):

Bước 1: Mở CMD và di chuyển vào thư mục src của dự án.

Bước 2: Gõ lệnh sau để biên dịch: javac quanlysinhvien/*.java

Bước 3: Gõ lệnh sau để chạy chương trình: java quanlysinhvien.MainApp

4. GHI CHÚ
